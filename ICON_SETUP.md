# アイコン・ファビコンの設定手順

## 画像ファイルの配置場所

以下のファイルを `app/` ディレクトリに配置してください：

### 1. ファビコン（必須）
```
app/favicon.ico
```
- サイズ: 16x16 または 32x32 または 48x48
- 形式: .ico
- 用途: ブラウザのタブに表示されるアイコン

### 2. アイコン画像（推奨）
```
app/icon.png
```
- サイズ: 512x512 以上推奨
- 形式: .png
- 用途: PWAやブラウザのブックマーク

### 3. Apple Touch Icon（推奨）
```
app/apple-icon.png
```
- サイズ: 180x180
- 形式: .png
- 用途: iPhoneやiPadでホーム画面に追加した際のアイコン

## 画像の準備方法

### オプション1: オンラインツールを使用
1. https://favicon.io/ にアクセス
2. ロゴ画像をアップロード
3. 各サイズのアイコンを自動生成
4. ダウンロードして配置

### オプション2: 手動でリサイズ
1. ロゴ画像を用意
2. 画像編集ソフト（Photoshop、GIMP、オンラインツールなど）でリサイズ
3. 各サイズで保存

## 配置手順（PC・Mac）

1. ロゴ画像ファイルを適切なサイズにリサイズ
2. ファイルを以下のパスに配置：
   ```
   /home/user/nexeed-blog/app/favicon.ico
   /home/user/nexeed-blog/app/icon.png
   /home/user/nexeed-blog/app/apple-icon.png
   ```

## 配置手順（iPhone/iPad）

iPhoneから直接配置することはできません。以下の方法をご検討ください：

### 方法1: PCを使用
1. iPhoneからPCに画像を転送（AirDrop、メールなど）
2. PCで上記の配置手順に従って配置

### 方法2: クラウド経由
1. iPhoneから画像をクラウドストレージにアップロード（iCloud、Google Drive、Dropboxなど）
2. PCでダウンロードして配置

### 方法3: Gitで管理
1. 別のブランチで画像をコミット
2. マージまたはプルリクエスト経由で取り込み

## 確認方法

ファイル配置後、以下のコマンドで確認：

```bash
ls -la /home/user/nexeed-blog/app/*.{ico,png}
```

次のビルド時に自動的に認識されます：

```bash
npm run build
```

## Next.jsの自動認識

Next.js 13以降では、`app/`ディレクトリに配置した以下のファイルが自動的に認識されます：

- `favicon.ico` → `/favicon.ico`
- `icon.png` → `/icon.png`
- `apple-icon.png` → `/apple-icon.png`

特別な設定は不要です！
