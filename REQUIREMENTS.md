# ブログサイト要件定義書

## プロジェクト概要

個人ブログサイト。投資、子育て、ITエンジニア、副業の4つのテーマを扱う。非技術者も記事を投稿できるMarkdownベースのブログシステム。

### サイト名
**NEXEED BLOG** (Next + Succeed = 次に成功する)

### カテゴリー
- 投資
- 子育て
- ITエンジニア
- 副業

## 技術スタック

### フロントエンド
- **フレームワーク**: Next.js (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **コンテンツ管理**: Markdown (Git管理)
- **デプロイ**: Vercel
- **ドメイン**: カスタムドメイン使用

### 推奨ライブラリ
- `gray-matter`: Markdownのフロントマター解析
- `remark` / `remark-html`: Markdown → HTML変換
- `remark-gfm`: GitHub Flavored Markdown対応
- `rehype-slug`: 見出しにIDを自動付与（目次用）
- `rehype-autolink-headings`: 見出しに自動リンク追加
- `rehype-highlight` / `rehype-prism`: コードシンタックスハイライト
- `next-seo`: SEO対策
- `date-fns`: 日付フォーマット

## 機能要件

### 1. 記事一覧ページ
- [ ] 記事をカード形式で一覧表示
- [ ] 各カードに以下の情報を表示
  - タイトル
  - 公開日
  - カテゴリー
  - アイキャッチ画像（任意）
  - 概要（excerpt）
- [ ] ページネーション機能（1ページあたりの表示件数: 10-12件程度）
- [ ] 最新記事順に表示

### 2. 記事詳細ページ
- [ ] Markdownで記述された記事本文を表示
- [ ] 記事メタ情報の表示
  - タイトル
  - 公開日・更新日
  - カテゴリー
- [ ] 目次（Table of Contents）
  - 長い記事には自動生成される目次を表示
  - 見出し（h2, h3）から自動生成
  - スムーズスクロール対応
- [ ] シンタックスハイライト対応
- [ ] レスポンシブな画像表示
- [ ] SNSシェアボタン
  - Twitter (X)
  - Facebook
  - はてなブックマーク
  - LINE
- [ ] 関連記事表示
  - 記事下部に同じカテゴリーの関連記事を表示（3-4件程度）
  - 最新記事または類似記事を表示

### 3. カテゴリー機能
- [ ] カテゴリー別記事一覧ページ（投資、子育て、ITエンジニア、副業）
- [ ] ヘッダーにカテゴリーナビゲーション

### 4. 検索機能
- [ ] キーワードによる記事検索
- [ ] タイトル、本文、カテゴリーを検索対象とする
- [ ] 検索結果一覧ページ
- [ ] クライアントサイド検索（静的サイト生成の場合）

### 5. プロフィールページ（About）
- [ ] 自己紹介・プロフィール情報
- [ ] アバター画像
- [ ] SNSリンク
- [ ] 経歴・実績など（任意）

### 6. 共通UI
- [ ] ヘッダー
  - サイトロゴ/タイトル
  - ナビゲーションメニュー（Home, カテゴリー, About）
  - 検索ボックス（モバイルはアイコン化）
- [ ] フッター
  - コピーライト
  - SNSリンク
  - Aboutページへのリンク
- [ ] レスポンシブ対応（モバイル、タブレット、デスクトップ）
- [ ] 1カラムレイアウト（サイドバーなし）

## コンテンツ構成

### ディレクトリ構造（案）
```
/content
  /posts
    /2024
      /01
        article-slug.md
        another-article.md
      /02
        ...
/public
  /images
    /posts
      article-image.png
```

### Markdownフロントマター（案）
```yaml
---
title: "記事タイトル"
date: "2024-01-15"
updated: "2024-01-20"  # 任意
category: "投資 | 子育て | ITエンジニア | 副業"
excerpt: "記事の概要文"
coverImage: "/images/posts/cover.png"  # 任意
---
```

## デザイン要件

### 参考サイト
- https://kurone43.com/

### レイアウト
- **1カラムレイアウト**（サイドバーなし）
- 記事一覧: **カード型**デザイン
- 中央揃え、ゆったりとした余白

### デザインの方向性
- シンプルで読みやすいデザイン
- 落ち着いた雰囲気
- レスポンシブ対応必須
- ダークモード対応（将来的に検討）

### カラースキーム
- **背景色**: 白（#FFFFFF）
- **テキスト色**: グレー系（#333333 - #666666）
- **アクセントカラー**: TBD（カテゴリーやリンクに使用）
- 全体的に落ち着いた配色

### タイポグラフィ
- **フォント**: ゴシック体
  - 日本語: Noto Sans JP、ヒラギノ角ゴシック、游ゴシック など
  - 英数字: Inter、Roboto など
- **余白**: ゆったりとした行間・段落間
