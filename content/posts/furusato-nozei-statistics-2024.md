# ふるさと納税の統計データ【2024年版】寄付額1.1兆円突破・利用者1,000万人超の実態

2023年度、ふるさと納税の全国寄付額が初めて**1兆円を突破**し、1兆1,175億円を記録しました。制度開始から15年以上が経過した現在、ふるさと納税は日本を代表する節税手段のひとつとして定着しています。本記事では、総務省が公表している最新の統計データをもとに、寄付額・利用者数の推移、年収別の節税効果、地域別の受入状況、そして2024年10月の制度改正が利用実態に与える影響を詳しく解説します。

## ふるさと納税制度の概要

### 制度の基本的な仕組み

ふるさと納税は、2008年（平成20年）に創設された寄付金控除制度です。個人が都道府県・市区町村（ふるさと）に寄付をすると、自己負担額2,000円を除いた全額が所得税・住民税から控除されます。

**制度の3つのポイント**:

1. **自己負担2,000円で返礼品がもらえる**: 寄付先の自治体から地域特産品などの返礼品を受け取れる
2. **税額控除で実質的な節税**: 上限内であれば寄付額から2,000円を引いた全額が翌年の税金から差し引かれる
3. **地域を選んで応援できる**: 出身地や関心のある地域を自由に選択できる

**税控除の仕組み（所得税・住民税）**:

| 控除の種類 | 控除額の計算 |
|-----------|------------|
| 所得税控除 | （寄付金額 − 2,000円）× 所得税率 |
| 住民税控除（基本分） | （寄付金額 − 2,000円）× 10% |
| 住民税控除（特例分） | 住民税所得割額 × 20%が上限 |

### ワンストップ特例制度

確定申告不要で手続きを完結できる「ワンストップ特例制度」が2015年に導入されたことで、利用のハードルが大幅に下がりました。

**ワンストップ特例の利用条件**:

- 確定申告が不要な給与所得者
- 年間の寄付先が5自治体以内
- 各自治体への申請書提出（翌年1月10日締切）

## 寄付額・利用者数の推移統計

### 全国寄付額の推移（2008〜2023年度）

ふるさと納税の寄付額は制度開始以来、右肩上がりで成長を続けています。

**寄付額の推移**:

| 年度 | 寄付総額 | 前年比 |
|------|---------|--------|
| 2008年度 | 81億円 | − |
| 2010年度 | 102億円 | +26.0% |
| 2015年度 | 1,652億円 | +3.8倍 |
| 2017年度 | 3,653億円 | +1.6倍 |
| 2019年度 | 4,875億円 | +33.4% |
| 2020年度 | 6,725億円 | +37.9% |
| 2021年度 | 8,302億円 | +23.4% |
| 2022年度 | 9,654億円 | +16.3% |
| **2023年度** | **1兆1,175億円** | **+15.8%** |

特筆すべきは2020年度の急成長で、新型コロナウイルス感染拡大の影響により在宅時間が増えたことで、ふるさと納税サイトの利用が急増しました。

### 利用者数の推移

**寄付件数・納税者数の推移**:

| 年度 | 寄付件数 | 推計利用者数 |
|------|---------|------------|
| 2015年度 | 726万件 | 約180万人 |
| 2018年度 | 2,322万件 | 約395万人 |
| 2020年度 | 3,489万件 | 約552万人 |
| 2021年度 | 4,447万件 | 約740万人 |
| 2022年度 | 5,184万件 | 約900万人 |
| **2023年度** | **5,895万件** | **約1,000万人超** |

2023年度は初めて推計利用者数が1,000万人を超えたとされています。日本の就業者数（約6,700万人）に対して約15%程度が利用している計算になります。

### 1件あたり平均寄付額

- **2023年度の1件あたり平均寄付額**: 約19,000円
- **利用者1人あたり平均寄付額**: 約100,000〜120,000円（複数件利用が一般的）

## 年収別の節税効果シミュレーション

### 控除上限額の目安（独身・共働き夫婦の場合）

ふるさと納税で全額控除される上限額（自己負担2,000円を除いた実質無料枠）は年収と家族構成によって異なります。

**独身または共働き（子どもなし）の場合の控除上限額目安**:

| 年収 | 控除上限額（目安） | 実質節税額（※） |
|------|-----------------|---------------|
| 300万円 | 約28,000円 | 約26,000円 |
| 400万円 | 約42,000円 | 約40,000円 |
| 500万円 | 約61,000円 | 約59,000円 |
| 600万円 | 約77,000円 | 約75,000円 |
| 700万円 | 約108,000円 | 約106,000円 |
| 800万円 | 約129,000円 | 約127,000円 |
| 1,000万円 | 約176,000円 | 約174,000円 |
| 1,500万円 | 約386,000円 | 約384,000円 |
| 2,000万円 | 約569,000円 | 約567,000円 |

※ 実質節税額 = 控除上限額 − 自己負担額2,000円（返礼品受取額は含まない）

**夫婦（共働き・子ども1人・16歳以上）の場合の控除上限額目安**:

| 年収 | 控除上限額（目安） |
|------|-----------------|
| 500万円 | 約49,000円 |
| 600万円 | 約69,000円 |
| 700万円 | 約86,000円 |
| 800万円 | 約120,000円 |
| 1,000万円 | 約163,000円 |

### 節税効果のシミュレーション例

**年収500万円・独身の場合（控除上限額61,000円）**:

1. 61,000円分のふるさと納税を実施
2. 受け取る返礼品の価値: 約18,300円相当（寄付額の30%）
3. 翌年の税金から差し引かれる額: 59,000円（＝61,000円 − 2,000円）
4. 実質的な支出: **2,000円で18,300円相当の返礼品を受け取れる**

**年収1,000万円・独身の場合（控除上限額176,000円）**:

1. 176,000円分のふるさと納税を実施
2. 受け取る返礼品の価値: 約52,800円相当（寄付額の30%）
3. 翌年の税金から差し引かれる額: 174,000円（＝176,000円 − 2,000円）
4. 実質的な支出: **2,000円で52,800円相当の返礼品を受け取れる**

- **参加率**: ほぼ100%（一部の特別区を除く）
- **運営ポータルサイト数**: 「さとふる」「ふるなび」「楽天ふるさと納税」「ふるさとチョイス」など大手5〜6社

## 2024年10月制度改正の影響

### 改正内容と背景

総務省は2024年10月1日から、ふるさと納税の経費算入ルールを厳格化しました。

**旧ルールと新ルールの比較**:

| 項目 | 旧ルール（〜2024年9月） | 新ルール（2024年10月〜） |
|------|--------------------|--------------------|
| 返礼品の上限 | 寄付額の30%以内 | 同左（変更なし） |
| 経費の上限（返礼品＋送料＋手数料等） | 寄付額の50%以内 | 寄付額の**50%以内** |
| ポータルサイト手数料の扱い | 経費に含まれない | **経費に含める** |

改正の最大のポイントは、**ふるさと納税ポータルサイトへの手数料が経費枠（50%以内）に含まれるようになった**ことです。手数料が従来の10〜15%かかっていたため、実質的に返礼品の割合が下がることになります。

### 改正後の利用者への影響

総務省が2024年度（改正後）に行った調査では、以下のような変化が報告されています。

**2024年度（4〜12月）の傾向**:

- **寄付件数**: 前年同期比 約 +8〜10%（伸びは鈍化）
- **返礼品の品質・量の変化**: 一部自治体で返礼品の内容を見直し（量の削減など）
- **利用者の行動変化**: ポータルサイト手数料の低い直接申込型・公式サイト型へ移行する利用者が増加

ふるさと納税制度は依然として利用者にとって有利な制度ですが、返礼品の実質的な価値が従来より若干下がるケースもあるため、複数のポータルサイトを比較して利用することが重要になっています。

### 返礼品カテゴリー別人気ランキング

各ふるさと納税ポータルサイトのデータをもとにした人気カテゴリーは以下の通りです。

**人気返礼品カテゴリー TOP10**（2023〜2024年度）:

| 順位 | カテゴリー | 特徴 |
|------|---------|------|
| 1 | 肉類（牛肉・豚肉・鶏肉） | 和牛ステーキ・焼肉セットが圧倒的人気 |
| 2 | 魚介類・海産物 | ホタテ・カニ・エビが上位 |
| 3 | 米 | ブランド米の大容量セットが人気 |
| 4 | 果物 | いちご・メロン・シャインマスカット等 |
| 5 | 野菜 | 旬の地場野菜セット |
| 6 | スイーツ・菓子 | 地域限定洋菓子・和菓子 |
| 7 | 飲料 | 地ビール・日本酒・ワイン |
| 8 | 加工食品 | ハム・ベーコン・漬物 |

食品カテゴリーが圧倒的な人気を誇り、特に「肉類」「魚介類」「米」の3カテゴリーで全寄付件数の約50%以上を占めると推計されています。

### 利用者の年代別傾向

ふるさと納税の利用は30〜50代の働き盛り世代を中心に広がっています。

**年代別利用傾向（推計）**:

| 年代 | 全体に占める割合 | 1人あたり平均寄付額 |
|------|--------------|-----------------|
| 20代 | 約8% | 約30,000円 |
| 30代 | 約25% | 約60,000円 |
| 40代 | 約30% | 約100,000円 |
| 50代 | 約25% | 約130,000円 |

40代・50代の高所得層が最も多く、控除上限額も大きいため1人あたりの寄付額が高い傾向にあります。

## 自治体の財政への影響：受入超過と流出超過

### 住民税の流出問題

ふるさと納税は、寄付先自治体の税収を増やす一方で、寄付者が居住する自治体の住民税を減少させる構造的な問題を抱えています。

**住民税の超過控除（流出）上位自治体（2022年度）**:

| 順位 | 自治体 | 住民税流出額（億円） |
|------|-------|------------------|
| 1 | 東京都世田谷区 | 約104億円 |
| 2 | 神奈川県横浜市 | 約285億円 |
| 3 | 大阪府大阪市 | 約202億円 |
| 4 | 愛知県名古屋市 | 約131億円 |
| 5 | 東京都港区 | 約68億円 |

大都市・富裕地区ほど住民税の流出額が大きく、横浜市・大阪市・名古屋市といった政令指定都市が上位を占めています。逆に地方の農漁業が盛んな自治体は大きな受入超過を実現しています。

## ふるさと納税とNISA・iDeCoとの組み合わせ戦略

### 税制優遇制度の比較

ふるさと納税は、NISA・iDeCoと組み合わせることで税金の最適化が図れます。ただし、iDeCoの控除額が大きくなると所得税の課税所得が下がり、ふるさと納税の控除上限額に影響することに注意が必要です。

**主要な税制優遇制度の比較**:

| 制度 | 税優遇の種類 | メリット | 注意点 |
|------|-----------|---------|-------|
| ふるさと納税 | 所得税・住民税の控除 | 返礼品という実質的な特典 | 上限額に注意・iDeCoと干渉 |
| NISA（つみたて枠） | 運用益・配当が非課税 | 長期の資産形成に有利 | 元本割れリスクあり |
| iDeCo | 掛金全額所得控除＋運用益非課税 | 節税効果が大きい | 60歳まで引き出せない |

### 賢い活用順序

1. **まず控除上限額を正確に把握**: 年収・家族構成をもとに計算ツールを利用
2. **iDeCoの掛金を決定**: iDeCo拠出後の課税所得をベースにふるさと納税の上限を再計算
3. **NISA（つみたて枠）で長期積立**: 年間120万円（月10万円）を目標に積立設定
4. **ふるさと納税を上限いっぱいまで活用**: 節税と同時に地域支援・お得な返礼品を享受

## まとめ：ふるさと納税の現在地と今後の展望

ふるさと納税は2023年度に寄付額1兆1,175億円・推計利用者1,000万人超という節目を迎えました。2024年10月の制度改正によって返礼品の実質価値が若干下がる部分もありますが、自己負担2,000円で高品質な地域の特産品を受け取れるメリットは健在です。

**今後の展望**:

1. **デジタル化の加速**: マイナンバーカード連携によるワンストップ手続きの簡略化
2. **返礼品の多様化**: 宿泊・体験型コンテンツや電子返礼品（電子書籍・音楽・ゲームなど）の拡大
3. **地域課題への活用**: 少子化対策・インフラ整備など特定目的の寄付型クラウドファンディング（ガバメントクラウドファンディング）の成長
4. **制度の持続可能性**: 大都市の税収流出問題への対応として、東京都などが独自の「都民ふるさと納税」的な仕組みを検討中

ふるさと納税は正しく活用すれば確実に「2,000円の実質負担で数万円相当の特典」を得られる、日本独自の節税・返礼品制度です。NISAやiDeCoと並ぶ家計の必須ツールとして、上限額の確認と計画的な活用を強くおすすめします。
