# Claude Codeを始めよう - インストールから初回起動まで完全ガイド

Claude CodeはAnthropicが開発した公式のCLIツールで、ターミナルから直接Claudeと対話しながらコーディングできる革新的な開発ツールです。この記事では、初めてClaude Codeを使う方向けに、セットアップから基本操作までを丁寧に解説します。

## Claude Codeとは

Claude Codeは、Anthropicが提供するAIアシスタント「Claude」をターミナルで直接使えるようにしたCLIツールです。コードの読み書き、ファイル操作、Git操作、テスト実行など、開発に必要な操作をAIがサポートしてくれます。

### 主な特徴

- **ターミナルネイティブ**: IDEに依存せず、どんな開発環境でも使える
- **コードベース理解**: プロジェクト全体を把握した上でコードの修正や追加ができる
- **ツール実行**: ファイルの読み書き、コマンド実行、Git操作などを自律的に行う
- **MCP対応**: Model Context Protocolで外部ツールと連携可能

## インストール方法

### 前提条件

Claude Codeを使うには以下が必要です：

- **Node.js 18以上**: `node --version` で確認できます
- **Anthropic APIキーまたはClaude Proプラン**: 認証に必要です

### npmでインストール

最も簡単なインストール方法はnpmを使う方法です：

```bash
npm install -g @anthropic-ai/claude-code
```

インストールが完了したら、バージョンを確認しましょう：

```bash
claude --version
```

## 初期設定

### 認証の設定

Claude Codeを初めて起動すると、認証設定が求められます：

```bash
claude
```

ブラウザが開き、Anthropicアカウントでの認証が行われます。Claude ProまたはTeamプランのユーザーは、追加のAPI料金なしで利用できます。

### 基本設定

Claude Codeには `/config` コマンドで設定を変更できます。主な設定項目：

- **テーマ**: ライト/ダークモードの切り替え
- **通知**: 処理完了時の通知設定
- **権限**: ツール実行の自動許可設定

## 基本的な使い方

### 対話モード

Claude Codeを起動すると対話モードに入ります。自然な日本語でリクエストを書くだけでOKです：

```
> このプロジェクトの構造を教えて

> src/components/Button.tsxにホバーエフェクトを追加して

> テストを実行して、失敗しているテストを修正して
```

### スラッシュコマンド

便利なスラッシュコマンドも用意されています：

- `/help` - ヘルプを表示
- `/clear` - コンテキストをクリア
- `/compact` - 会話を要約してコンテキストを節約
- `/config` - 設定を変更

## まとめ

Claude Codeは、AIを活用した開発の新しいスタンダードとなるツールです。まずはインストールして、小さなタスクから試してみましょう。次回の記事では、より実践的なTipsを紹介していきます。
