ゲーム業界・イラスト・CG・eスポーツ分野でプロを目指したい方にとって、学校選びは非常に重要です。「どの専門校が自分に合っているか」を判断するうえで、まず無料の資料請求（パンフレット請求）から始めることをおすすめします。

## バンタンゲームアカデミーとは

バンタンゲームアカデミーは、ゲーム・アニメ・イラスト・CG・eスポーツ分野のクリエイター・プロフェッショナルを育成する専門スクールです。

### 基本情報

| 項目 | 内容 |
|------|------|
| 正式名称 | バンタンゲームアカデミー 専門部 |
| 運営会社 | 株式会社バンタン（KADOKAWAグループ・ドワンゴ子会社） |

### バンタンとは

株式会社バンタンは、1965年に創立した「バンタンデザイン研究所」を源流とする教育機関グループです。現在はKADOKAWAグループのドワンゴ子会社として、ゲーム・ファッション・料理・美容など幅広いクリエイター教育を展開しています。

**グループの主な教育機関:**

- バンタンゲームアカデミー（ゲーム・イラスト・CG・eスポーツ）
- バンタンデザイン研究所（デザイン・ファッション）
- バンタン調理芸術高等部（料理）
- バンタン高等学院（通信制高校連携）

## 学部・コース構成

バンタンゲームアカデミー専門部では、4学部・複数のコースが設置されています。

### ゲーム学部

ゲーム制作に必要なプログラミング・デザイン・企画などのスキルを学ぶ学部です。

**主な専攻・コース:**

| 専攻名 | 修業年限 | 習得スキル |
|--------|---------|-----------|
| ゲームプログラマー専攻 | 2〜4年制 | Unity・Unreal Engine・C++・ネットワーク開発 |
| ゲームプランナー専攻 | 2〜3年制 | ゲームデザイン・企画・レベルデザイン |
| キャラクターデザイン専攻 | 2〜4年制 | 2DCG・3DCG・キャラクターイラスト |
| 3DCGアニメーション専攻 | 2〜4年制 | 3DCGモデリング・アニメーション・VFX |
| ゲームグラフィック専攻 | 2〜3年制 | UIデザイン・テクスチャー・グラフィック |

**使用ツール（例）:**

- Unity・Unreal Engine（ゲームエンジン）
- Maya・Blender・ZBrush（3DCGソフト）
- Photoshop・Illustrator・Clip Studio Paint（2DCGソフト）
- Visual Studio・C++・C#・Python（プログラミング）

### eスポーツ学部

日本でも成長著しいeスポーツ分野に特化した学部です。プロゲーマーとしてのスキルから、ストリーマー・コンテンツクリエイターまで幅広く学べます。

**主な専攻・コース:**

| 専攻名 | 修業年限 | 目指す職業 |
|--------|---------|-----------|
| プロゲーマー専攻 | 2年制 | プロゲーマー・eスポーツアスリート |
| eスポーツクリエイター専攻 | 3年制 | ストリーマー・動画クリエイター・実況者 |
| eスポーツビジネス専攻 | 2〜3年制 | チームマネージャー・イベントプロデューサー |

**学習内容（eスポーツ系）:**

- ゲームスキルの強化（戦略・チームワーク）
- 動画編集・配信技術（OBS・Adobe Premiere Pro）
- ゲームビジネスの仕組み・マネジメント
- SNS・YouTubeを活用したコンテンツ制作

### イラスト学部

プロのイラストレーター・コンセプトアーティストを目指す学部です。2026年4月開講のコースも含まれています。

**主な専攻:**

- キャラクターイラスト専攻
- コンセプトアート専攻
- 背景・環境アート専攻

**学習内容（イラスト系）:**

- デッサン・色彩・構図などの基礎
- デジタルイラスト制作（Clip Studio Paint・Photoshop）
- 商業キャラクターデザインの実践
- ポートフォリオ制作・作品審査会への参加

### オンライン学部

2026年4月開講予定の新学部です。通学が難しい地方在住者や社会人向けに、オンラインで質の高い教育を提供します。

**特徴:**

- 全授業がオンラインで受講可能
- 年間学費：118万円（通学コースより低コスト）
- 通学コースと同等のカリキュラム内容
- 録画視聴・ライブ授業の両方に対応

### 大学部

専門スキルを学びながら「大学卒業資格」も同時に取得できる、新しい進路の形です。

**連携大学（2026年現在）:**

| 連携大学 | 学習方法 | 取得できる学位 |
|---------|---------|--------------|
| 京都芸術大学 通信教育部 | 通信制 | 学士（芸術） |
| 星槎大学 | 通信制 | 学士（共生科学） |
| ZEN大学 | 通信制 | 学士 |

**大学部のメリット:**

1. 専門スキルと大卒資格を同時取得
2. 就職活動で「大卒」要件を満たせる
3. 大学院進学の選択肢が広がる
4. 奨学金（日本学生支援機構）が利用可能（連携大学経由）

## 学費・費用の概要

バンタンゲームアカデミーは無認可校であるため、学費については事前にしっかりと確認しておく必要があります。

### 年間学費の目安（2026年度）

| 学部・コース | 修業年限 | 年間学費目安 |
|------------|---------|------------|
| ゲーム学部（4年制） | 4年 | 141万円 |
| ゲーム学部（3年制） | 3年 | 141万円 |
| ゲーム学部（2年制） | 2年 | 146万円 |
| eスポーツ学部 | 2〜3年 | 141〜146万円 |
| イラスト学部 | 2〜4年 | 141〜146万円 |
| オンライン学部 | 2〜3年 | 118万円 |
| 大学部 | 2〜4年 | 124万円＋連携大学費用 |

※入学金（10万円）は初年度に別途必要
※教材費・学友会費（年間1万円）は別途必要
※大学部は連携大学の学費が別途発生

### 学費以外の諸費用

入学時および在学中には、学費以外にも以下の費用がかかります。

**主な諸費用:**

- **入学金**: 10万円（初年度のみ）
- **学友会費**: 年間1万円
- **教材費**: 専攻によって異なる（PC・ソフトウェア等）
- **海外研修費**: 希望者のみ（任意）

**ソフトウェア・PCについて:**

バンタンゲームアカデミーでは、校内の設備を使用して授業を受けるため、必ずしも個人のPCが必要なわけではありません。ただし、自宅学習や課題制作のために、対応スペックのPCを用意することを推奨しています。詳細はパンフレットまたは個別相談で確認できます。

### 奨学金・学費支援について

バンタンゲームアカデミー専門部は**各種学校（無認可校）**であるため、日本学生支援機構（JASSO）の奨学金は原則として利用できません。ただし、以下の方法で学費の負担を軽減できる可能性があります。

**学費支援の選択肢:**

1. **大学部経由の奨学金**: 連携大学（京都芸術大学・星槎大学・ZEN大学）に在籍することで、JASSOの奨学金が利用可能になるケースがあります。詳細は各大学の規定を確認してください。

2. **バンタン独自の特待生制度**: 選考により学費が減免される制度があります。詳細はパンフレットまたは個別相談で確認できます。

3. **教育ローン（民間）**: 各種金融機関の教育ローンを利用できます。

4. **分割払いオプション**: 学費の分割納付が可能な場合があります。詳細は入学手続き時に確認してください。

### 就職サポートプログラム

**在学中から始まるサポート体制:**

1. **専任アドバイザーによる個別就職相談**
   - クラス担任とは別に、就職担当スタッフが個別サポート

   - ゲーム・エンタメ企業の業界研究会

4. **スカウト展（ポートフォリオ審査会）**
   - 企業の採用担当者に対して直接作品をプレゼン

5. **東京ゲームショウへの出展**
   - 学生作品を世界最大のゲームイベントで展示
   - 業界関係者に作品を見てもらえる機会

**無認可校であること:**

- バンタンゲームアカデミー専門部は「各種学校」であり、専門学校（専修学校専門課程）ではありません
- 「専門士」「高度専門士」の称号は取得できません（連携大学部を除く）
- 就職活動で「学歴」として扱われる範囲が異なる場合があります

**カリキュラムの密度:**

- 実践的な授業が多く、課題量も多いため、ついていくには相応の努力が必要との声もあります
