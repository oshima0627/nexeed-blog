# Claude Code 2026年最新アップデートまとめ

2026年に入り、AnthropicのAIコーディングアシスタント **Claude Code** は目覚ましい進化を遂げています。本記事では、2026年3月時点における主要な新機能・アップデートを、公式CHANGELOGおよび複数の技術メディアの報告をもとに整理・解説します。

---

## 1. 音声モード（Voice Mode）の追加

**リリース日：2026年3月3日**

Claude Codeにネイティブの音声入力機能が追加されました。`/voice` コマンドで有効化し、**スペースバーを押しながら話す「プッシュトゥトーク方式」**で操作できます。

### 対応プランと料金

| プラン | 追加料金 |
|--------|----------|
| Pro | なし |
| Max | なし |
| Team | なし |
| Enterprise | なし |

### 対応言語

英語・日本語を含む**20言語**に対応。2026年3月の追加言語にはロシア語・ポーランド語・トルコ語・オランダ語・ウクライナ語などが含まれます。

> **注意**：リリース時点では全ユーザーの約5%が対象で、段階的に拡大予定とのことです（AI総合研究所の報告より）。

---

## 2. Remote Control（リモートコントロール）

**リリース日：2026年2月25日（リサーチプレビュー）**

パソコンのターミナルで実行中のClaude Codeのローカルセッションを、**スマートフォン・タブレット・別のPCのブラウザ**から遠隔操作できる機能です。

### 主な特徴

- `claude.ai/code` またはClaude モバイルアプリから接続可能
- コードの実行はローカルマシン上で行われるため、MCP サーバーやローカルファイルへのアクセスを維持
- デバイス間で会話履歴がリアルタイムに同期
- 移動中や外出先からでも作業を継続できる

### 競合との差別化

「ローカル実行 × フル環境アクセス × モバイル操作」の三拍子を兼ね備えた点が、現時点でRemote Control独自の強みとされています（複数の技術レビューより）。

---

## 3. ultrathinkキーワードの復活

**バージョン：v2.1.68**

ユーザーからの要望に応え、「**ultrathink**」キーワードが再導入されました。プロンプトに `ultrathink` と含めるだけで、そのターンのみ **high effort（高処理能力）モード**が有効になります。

### 合わせて知っておくべき変更点

- **Opus 4.6 のデフォルト effort が `medium` に変更**：品質を保ちながら効率的にコーディングタスクを処理するバランスが調整されています
- `ultrathink` は特に複雑な設計・リファクタリングなどで効果的です

---

## 4. /loop コマンドとスケジュールタスク

**バージョン：v2.1.71**

プロンプトを定期実行できる `/loop` コマンドとスケジュールタスク機能が追加されました。

### 使い方の例

```
/loop 30m /security-review
```
→ 30分ごとにセキュリティレビューを自動実行

| 種別 | 特徴 |
|------|------|
| CLI版 | セッションスコープ（セッション中のみ有効） |
| Desktop版 | 永続的なスケジュール設定が可能 |

## 5. Claude Code Desktop の強化

**リリース日：2026年2月20日**

デスクトップアプリに以下の機能が追加されました：

### アプリプレビュー

開発サーバーを起動し、デスクトップ上でアプリの動作を直接プレビュー。コンソールログを読み取り、エラーを自動修正します。

### コードレビュー

プッシュ前にClaude Codeがインラインコメントでバグや問題点を指摘します。

### CI/PR のバックグラウンド処理

PRのCIパイプラインを監視し、失敗時に自動修正を試みます。

---

## 6. バージョン 2.1.80 の主な変更点

最新バージョン（2026年3月時点）では以下の改善が行われました：

### 新機能

- **ステータスラインにレート制限情報を表示**：5時間・7日間ウィンドウでのClaude.ai使用量をパーセンテージとリセット時刻とともに確認可能
- **`--channels` フラグ（リサーチプレビュー）**：MCPサーバーからセッションにメッセージをプッシュできる
- **`effort` フロントマター対応**：スキル・スラッシュコマンド呼び出し時のモデルeffortレベルをオーバーライド可能

### パフォーマンス改善

- 大規模リポジトリ（ファイル数250,000超）での起動メモリ使用量を**約80MB削減**
- 大規模gitリポジトリでの `@` ファイルオートコンプリートの応答速度向上

### バグ修正

- `--resume` 実行時の並列ツール結果の欠落問題を修正
- 音声モードのWebSocket接続失敗（CloudflareのTLSフィンガープリント検出）を修正

---

## 7. セキュリティ修正（重要）

**バージョン：v2.1.69**

`acceptEdits` モード使用時に、シンボリックリンク経由でワーキングディレクトリ外のファイルを作成できてしまう脆弱性が修正されました。

**v2.1.69以降へのアップデートが強く推奨されています。**

アップデートコマンド：
```bash
npm update -g @anthropic-ai/claude-code
```

## 2026年のClaude Code：「エージェント型開発ツール」への進化

2026年のClaude Codeは、単なるコード補完ツールから**「自律的に開発タスクを遂行するエージェント」**へと大きく進化しています。

音声操作・リモート操作・高度な推論（ultrathink）・定期タスク自動化といった機能群が揃い、開発者の働き方を根本から変える可能性を秘めたツールとなっています。
