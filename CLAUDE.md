# Claude Code Blog — Claude 作業ルール

このファイルはClaude Codeが常に参照するルール定義ファイルです。
全ての作業において以下のルールを遵守してください。

---

## サイト概要

- **サイト名**: Claude Code Blog
- **テーマ**: Claude Code（Anthropic公式CLIツール）に特化した日本語情報ブログ
- **カテゴリー**: 入門ガイド / Tips・活用術 / MCP・拡張機能 / 開発事例 / ニュース

---

## 記事作成ルール

### 1. 情報の正確性
- Claude Codeの公式ドキュメント・GitHubリポジトリを情報源とすること
- Anthropicの公式発表を最優先で参照する
- 不確かな情報は記事に含めない
- バージョン固有の情報には対象バージョンを明記する

### 2. 記事フォーマット
- Markdownフロントマターのフォーマットを遵守する
- **`date` フィールドには記事ファイルの実際の作成日時を ISO 8601 形式（`"YYYY-MM-DDTHH:MM:SS"`）で記載すること**
- カテゴリーは「入門ガイド」「Tips・活用術」「MCP・拡張機能」「開発事例」「ニュース」の5つのみ
- excerptは100〜150字程度の概要文
- コードブロックには言語を指定する

### 3. 禁止事項
- AIが生成した推測を事実として記載することを禁止
- 古い・非推奨の情報を最新として紹介することを禁止
- 誇張表現・センセーショナルな見出しを避ける

---

## カバー画像のルール（必須）

新しく記事を作成する際は、`public/` フォルダ内の **未使用のUnsplash画像** の中から記事のテーマに合った画像を選ぶこと。

### 手順
1. `Glob` ツールで `public/*-unsplash.jpg` の一覧を取得する
2. `Grep` ツールで `content/posts/**/*.md` の `coverImage` フィールドを検索し、すでに使用済みの画像を特定する
3. **未使用の画像**の中から記事内容に最も関連する画像を選ぶ
4. フロントマターの `coverImage` に `"/<ファイル名>"` 形式で設定する

### 選定基準
- 記事の具体的なトピック・内容と視覚的に一致する画像を選ぶこと
- テクノロジー系の画像（コード、ターミナル、PC）を優先する
- 同じ画像を複数の記事で使い回さない
