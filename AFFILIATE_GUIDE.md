# アフィリエイトリンク最適配置ガイド - NEXEED BLOG

## 📊 調査結果サマリー

### コンバージョン率（CVR）の目安
- **業界平均**: 1%〜3%
- **初心者目標**: 1%
- **上級者**: 3%〜5%

### クリック率向上のポイント
- 記事冒頭・本文中・末尾の**3箇所**に配置が効果的

- **マイクロコピー**（短い説明文）の活用が重要

## 🎯 NEXEED BLOG 最適配置プラン

### 1. 記事詳細ページ（優先度：★★★）

#### A. 記事冒頭エリア（ファーストビュー直下）
**配置場所**: 目次の直前または直後
**推奨形式**: カード型バナー
**理由**:
- ページを開いた直後の高い注目度
- 記事を読む前の「解決策を探している」状態でアプローチ
- CVR: 約2%〜3%期待

```
┌─────────────────────┐
│ 記事タイトル        │
│ 日付・カテゴリー    │
├─────────────────────┤
│ 💡 おすすめ商品     │ ← ここ！
│ [CTAボタン]         │
├─────────────────────┤
│ 目次                │
│ 記事本文            │
└─────────────────────┘
```

**カテゴリー別推奨リンク**:
- **投資**: 証券口座開設、投資信託、マネー本
- **子育て**: 育児グッズ、教育サービス、子育て本
- **ITエンジニア**: プログラミング学習、技術書、開発ツール
- **副業**: クラウドソーシング、ビジネス書、ツール

---

#### B. 記事本文中（コンテキスト連動）
**配置場所**: 商品・サービス紹介セクションの直後
**推奨形式**: テキストリンク + ボタン
**理由**:
- 具体的な説明直後で購買意欲が高い
- 自然な流れで違和感なし
- CVR: 約3%〜5%期待（最も高い）

```markdown
## おすすめの投資信託

eMAXIS Slim 全世界株式は、初心者に最適な...

【ここに詳細説明】

> 💡 **今すぐ始めるなら**
> SBI証券なら取引手数料無料！
> [無料で口座開設する →]

次のセクションへ...
```

---

#### C. 記事末尾（クロージング）
**配置場所**: SNSシェアボタンの直前
**推奨形式**: まとめ型CTA
**理由**:
- 記事を読み終えた「納得した」タイミング
- 行動を促す最後のチャンス
- CVR: 約1.5%〜2%期待

```
┌─────────────────────┐
│ 記事本文（最後）    │
├─────────────────────┤
│ まとめ              │
│ この記事で紹介した  │
│ おすすめツール3選   │
│ [1] [2] [3]         │ ← ここ！
├─────────────────────┤
│ SNSシェアボタン     │
│ 関連記事            │
└─────────────────────┘
```

---

### 2. サイドバー風エリア（優先度：★★☆）

**現状**: 1カラムレイアウトのためサイドバーなし
**提案**: 記事下部に「おすすめ情報」セクションを追加

**配置場所**: 関連記事の下
**推奨形式**: カテゴリー別おすすめリンク集
**理由**:
- 関連記事を見た後の「もっと詳しく知りたい」に対応
- CVR: 約0.5%〜1%

```
┌─────────────────────┐
│ 関連記事            │
├─────────────────────┤
│ このカテゴリーの    │
│ おすすめサービス    │
│ ・証券会社比較      │ ← ここ！
│ ・投資本ランキング  │
└─────────────────────┘
```

---

### 3. 専用ランディングページ（優先度：★★★）

**新規作成推奨**: `/recommended` または `/tools`
**内容**: カテゴリー別おすすめ商品・サービス一覧
**理由**:
- SEOで「おすすめ」「比較」などで集客可能
- じっくり比較検討するユーザー向け
- CVR: 約2%〜4%期待

**ページ構成例**:
```
/recommended
  /investment    → 投資関連のおすすめ
  /parenting     → 子育て関連のおすすめ
  /engineering   → エンジニア向けツール
  /side-business → 副業サービス比較
```

---

### 4. フッター（優先度：★☆☆）

**配置場所**: フッターリンクエリア
**推奨形式**: テキストリンク
**理由**:
- 全ページ共通で露出
- CVR: 約0.2%〜0.5%（低いが継続的）

```
┌─────────────────────┐
│ © 2026 NEXEED BLOG  │
│ About | おすすめ    │ ← ここ！
└─────────────────────┘
```

## 🎨 デザインガイドライン

### 推奨ボタンデザイン

#### 1. プライマリCTA（最重要リンク）
```css
背景色: #22C55E（緑）または #1E40AF（ブランドブルー）
テキスト: 白
形状: 角丸（8px）
サイズ: 大（padding: 16px 32px）
```

```css
テキスト: #1E40AF
形状: 角丸（8px）
```

## 🔧 技術実装方針

### 1. アフィリエイトリンクコンポーネント
```tsx
<AffiliateCard
  title="SBI証券"
  description="初心者におすすめ！手数料無料"
  link="https://..."
  badge="おすすめ"
  category="investment"
/>
```

### 2. インライン挿入
```markdown
記事内で [[affiliate:sbi-securities]] と書くと
自動的にアフィリエイトカードに変換
```

### 3. 管理方法
- `/data/affiliates.json` でリンク一元管理
- カテゴリー別に整理
- リンク切れチェック機能

## ⚠️ 注意事項

### 法規制・ガイドライン
1. **ステマ規制対応** (2023年10月施行)
   - 「PR」「広告」「アフィリエイトリンク」表記必須
   - 記事冒頭または各リンク付近に明記

2. **ASP規約遵守**
   - 各ASPの利用規約を確認
   - 禁止事項（メール直リンクなど）を遵守

3. **ユーザビリティ**
   - 過度な広告は避ける
   - コンテンツ品質を最優先
